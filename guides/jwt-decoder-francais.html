Décoder un token JWT en ligne

Ce guide explique jwt décodeur français — comment décoder un JSON Web Token, lire le header et le payload, et diagnostiquer les erreurs courantes.

Réponse rapide

Utilisez notre Décodeur JWT pour inspecter le token, puis le Validateur JWT pour vérifier la signature. Tout s'exécute dans votre navigateur.

Qu'est-ce qu'un JWT ?

Un JWT (RFC 7519) comporte trois parties Base64URL : header, payload (claims) et signature.

Étapes

  1. Copiez le JWT depuis votre application ou les outils de développement
  2. Collez-le dans le Décodeur JWT
  3. Vérifiez exp, iss, aud et alg
  4. Vérifiez la signature avec le Validateur JWT
  5. Utilisez le Debugger JWT pour l'analyse détaillée

Sécurité

Décoder n'est pas authentifier. Validez toujours la signature et les claims côté serveur.
